Make a static website
 - index.html
   - homepage
   - should describe what the webpage is
   - hyperlinks to the following:
     - aboutme.html
       - your bio
         - name, degree, born
     - hobby.html
       - page dedicated to something you like
     - link to a "cool" SFW website to discuss

Required tags:
- p
- table
- ul
- ol
- img

At least one of each:
 - inline CSS
 - internal CSS
 - external CSS
All pages should have their own CSS file.

Optional - leverage Bootstrap
 - Use at least 3 different CSS selector types, e.g.
	el
	el > el2
	el ~ el3
